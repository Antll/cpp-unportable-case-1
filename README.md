# Usage     
`make [CLANG=1|0] [PLATFORM=linux64|linux32]`  

`CLANG=0|1`  
Compile with Clang compiler. Otherwise compile with g++.  
    
`PLATFORM=linux64|linux32`  
Compile against x64 of x32 GNU/Linux. x64 is default.
